# 註冊表單



## 自訂錯誤條件


1.  **手機號碼 (`patternMismatch`)**
    -   **條件**：當輸入值不符合表示式時。
    -   **msg**：「請輸入有效的 10 碼手機號碼」。

2.  **密碼 (`customError`)**
    -   **條件**：當密碼未同時包含英文字母和數字時。
    -   **msg**：密碼需包含英文及數字。

3.  **確認密碼 (`customError`)**
    -   **條件**：當此欄位的值與「密碼」欄位的值不一致時。
    -   **msg**：兩次輸入的密碼不一致。

4.  **興趣標籤 (`customError`)**
    -   **條件**：當沒有勾選任何一個興趣標籤時。
    -   **msg**：請至少選擇一項興趣。

---

## 加分


1.  **密碼強度條**
    -   根據密碼的長度、是否包含大小寫字母數字符號，強度分為弱/中/強。

2.  **localStorage**
    -   表單中的所有輸入內容都會存到 `localStorage` 中。
    -   送表單後暫存資料會自動清除。

3.  **重設按鈕**
    -   點擊後會清除所有欄位的內容，並清除 `localStorage` 中的暫存資料，將表單恢復到初始狀態。
